Code Tag

The code tag is used to represent a piece of code, such as a function or method name:

For example, the following code uses the console.log() method to print a message to the console:

console.log("Hello World!");

Keyboard Tag

The kbd tag is used to represent input that is typically entered using a keyboard:

For example, to save a file in a text editor, you might need to press the Ctrl + S keys on your keyboard.

Sample Tag

The samp tag is used to represent a sample output from a program or script:

For example, if you ran the following script:

echo "Hello World!";

The output would be:

Hello World!

Variable Tag

The var tag is used to represent a variable in a program or script:

For example, in the following code, the variable name is defined and assigned a value:

var name = "John";

Preformatted Tag

The 

pre
tag is used to represent preformatted text, which preserves whitespace and formatting:

For example, the following text will be displayed exactly as written, with line breaks and indentations preserved:

  This is a
  preformatted
  piece of text.

Listing Tag

The

listing tag is used to represent a block of computer code or other preformatted text:

For example, the following code is written in JavaScript:

function greet(name) { console.log("Hello " + name + "!"); } greet("John");
